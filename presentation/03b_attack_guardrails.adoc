public class PromptInjectionGuardrail implements InputGuardrail {

    private static final List<String> BLACKLIST = List.of(
        "ignore previous instructions",
        "system override",
        "delete database",
        "reveal system prompt"
    );

    private InputGuardrailResult checkBlacklist(String text) {
        String lower = text.toLowerCase();
        for (String phrase : BLACKLIST) {
            if (lower.contains(phrase)) {
                return fatal("Security Violation: Potential prompt injection detected.");
            }
        }
        return success();
    }
}
----

* *Blacklists* catch "ignore instructions" attacks
* *Length checks* block context-window padding ("prompt stuffing")
* *PII redaction* scrubs emails, SSNs, credit cards from responses

_"You can't jailbreak what you can't reach. 50,000 characters for a one-page invoice? Something's wrong."_
