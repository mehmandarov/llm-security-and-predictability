==== Tech Stack

* *Java 21* (LTS) • *LangChain4j* • *Ollama*
* *Jakarta Validation* • *JUnit 5 & Mockito*
* *Maven Failsafe* for integration tests

==== Running the Demo

```bash
# Mocked tests (fast, no LLM needed)
mvn test

# Live tests against Ollama (the real chaos)
ollama serve && ollama pull gemma4:e2b
mvn verify
```
